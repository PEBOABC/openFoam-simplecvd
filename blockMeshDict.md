## 整体结构解析

这是一个OpenFOAM的`blockMeshDict`文件，用于生成**二维轴对称管式反应器**的计算网格，是CVD（化学气相沉积）模拟的基础。

---

## 文件头（Header）

```cpp
/*--------------------------------*- C++ -*----------------------------------*
| =========                 |                                                 |
| \\      /  F ield         | OpenFOAM: The Open Source CFD Toolbox           |
|  \\    /   O peration     | Version:  13                                    |
|   \\  /    A nd           | Website:  www.openfoam.com                      |
|    \\/     M anipulation  |                                                 |
\*---------------------------------------------------------------------------*/
FoamFile
{
    version     2.0;
    format      ascii;
    class       dictionary;
    object      blockMeshDict;
}
```

**意义**：标准的OpenFOAM文件头，声明这是一个字典文件，用于`blockMesh`网格生成工具。

---

## 单位缩放

```cpp
convertToMeters 0.001;
```

**意义**：所有坐标值乘以0.001才是实际米制单位。  
- 坐标中`200` → 实际长度 `200 × 0.001 = 0.2米（200毫米）`
- 坐标中`20` → 实际高度 `0.02米（20毫米）`
- 坐标中`0.1` → 实际厚度 `0.0001米（0.1毫米）`

**设计意图**：反应器尺寸为 **长200mm × 高20mm × 厚0.1mm**（二维模拟需要很小的厚度方向）。

---

## 顶点定义（Vertices）

```cpp
vertices
(
    // 底面四点（逆时针）
    (0 0 0)      // 0 入口底部
    (200 0 0)    // 1 出口底部
    (200 20 0)   // 2 出口顶部
    (0 20 0)     // 3 入口顶部
    
    // 顶面四点（与底面对应）
    (0 0 0.1)    // 4 入口底部后
    (200 0 0.1)  // 5 出口底部后
    (200 20 0.1) // 6 出口顶部后
    (0 20 0.1)   // 7 入口顶部后
);
```

**意义**：定义六面体的8个顶点坐标 `(x, y, z)`：

| 坐标轴 | 含义 |
|--------|------|
| x | 流动方向（0→200 mm） |
| y | 高度方向（0底部→20 mm顶部） |
| z | 厚度方向（0前→0.1 mm后，用于2D模拟） |

**设计意图**：创建一个矩形管道，气体从左侧（x=0）流入，从右侧（x=200）流出，顶部（y=20）是加热基底。

---

## 块定义（Blocks）

```cpp
blocks
(
    hex (0 1 2 3 4 5 6 7) (100 20 1) simpleGrading (1 1 1)
);
```

**意义**：
- `hex (0 1 2 3 4 5 6 7)`：使用8个顶点构建一个六面体块
- `(100 20 1)`：网格划分
  - x方向：100个网格
  - y方向：20个网格
  - z方向：1个网格（2D模拟）
- `simpleGrading (1 1 1)`：均匀网格，无加密

**设计意图**：生成 **100×20×1 = 2000个六面体网格单元**，足以捕捉CVD过程中的流动和传质边界层。

## 边界定义（Boundary）

### 入口（inlet）

```cpp
inlet
{
    type patch;
    faces
    (
        (0 3 7 4)    // 入口面：x=0
    );
}
```

**意义**：定义气体入口边界。面由顶点 `0→3→7→4` 组成（逆时针方向）。这是 **x=0 的平面**。

### 出口（outlet）

```cpp
outlet
{
    type patch;
    faces
    (
        (1 5 6 2)    // 出口面：x=200
    );
}
```

**意义**：定义气体出口边界。面由顶点 `1→5→6→2` 组成。这是 **x=200 的平面**。

### 基底（substrate）- 关键边界

```cpp
substrate
{
    type wall;
    faces
    (
        (3 2 6 7)    // 顶部壁面：y=20（基底）
    );
}
```

**意义**：定义加热基底表面，类型为壁面。面由顶点 `3→2→6→7` 组成。这是 **y=20 的顶面**，将设置高温和浓度边界条件模拟CVD沉积。

### 底部壁面（bottomWall）

```cpp
bottomWall
{
    type wall;
    faces
    (
        (0 1 5 4)    // 底部壁面：y=0
    );
}
```

**意义**：定义反应器底部壁面，类型为壁面。面由顶点 `0→1→5→4` 组成。这是 **y=0 的底面**。

### 前后边界（front & back）- 2D关键

```cpp
front
{
    type empty;
    faces
    (
        (0 1 2 3)    // 前面：z=0
    );
}
back
{
    type empty;
    faces
    (
        (4 5 6 7)    // 后面：z=0.1
    );
}
```

**意义**：`type empty` 是OpenFOAM中**二维模拟的关键设置**，表示该方向没有流动梯度。这两个面垂直于z轴，被标记为"empty"后，求解器会自动将模拟简化为二维。

## 整体设计意图总结

| 元素 | 意义 | 在CVD中的角色 |
|------|------|--------------|
| 长200mm | 反应器长度 | 提供足够流动发展距离 |
| 高20mm | 反应器高度 | 确定边界层厚度 |
| 厚0.1mm | 二维厚度 | 实现2D模拟 |
| 100×20网格 | 网格分辨率 | 分辨浓度边界层 |
| 入口 | 气体流入 | 载气+前驱体A进入 |
| 出口 | 气体流出 | 废气排出 |
| 基底（顶面） | 加热表面 | CVD沉积发生位置 |
| 底部壁面 | 冷壁 | 无反应 |
| 前后边界 | empty类型 | 实现2D模拟 |

这个网格为后续设置边界条件（基底高温1200K、入口含1%前驱体A）提供了几何基础，是整个CVD模拟的第一步。
